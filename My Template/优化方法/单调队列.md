# 单调队列模板
可以以 $O(n)$ 复杂度解决滑动窗口问题，求所有长度为k的子区间的最大元素。
```cpp
deque<int> q;
vector<int> res;

signed main() {
    cin >> n >> k;
    for (int i = 1; i <= n; i++) cin >> a[i];

    for (int i = 1; i <= n ;i++) {
        while (!q.empty() && q.front() < i - k + 1) q.pop_front();
        while (!q.empty() && a[q.back()] < a[i]) q.pop_back();
        q.push_back(i);
        if (i >= k) res.push_back(a[q.front()]);
    }
}
```
