# 单调栈模板
可以以 $O(n)$ 复杂度求每个元素某侧第一个比它大的元素的下标或值。
```cpp
stack<int> st;
vector<int> ne_r;
vector<int> a;

signed main() {
    cin >> n;
    ne_r.resize(n), a.resize(n);

    for (int i = 0; i < n; i++) {
        int h; cin >> h;
        while (!st.empty() && h >= st.back()) st.pop();
        st.push(h);
    }
}
```
