# 二分模板
## 二分查找
```cpp
int find1(int x) {
    int l = 1, r = n;
    while (l < r) {
        int mid = (l + r) / 2;
        if (a[mid] >= x) r = mid;
        else l = mid + 1;
    }
}

lower_bound(a + 1, a + 1 + n, x);

int find2(int x) {
    int l = 1, r = n;
    while (l < r) {
        int mid = (l + r + 1) / 2;
        if (a[mid] <= x) l = mid;
        else r = mid - 1;
    }
}

upper_bound(a + 1, a + 1 + n, x);
```
## 二分答案
```cpp
int find1(int x) {
    int l = 1, r = inf;
    while (l < r) {
        int mid = (l + r) / 2;
        int c = check(mid);
        if (c == x) return mid;
        else if (c > x) r = mid - 1;
        else l = mid + 1;
    }
}
```
