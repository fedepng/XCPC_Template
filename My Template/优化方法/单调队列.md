# 单调队列模板
可以以 $O(n)$ 复杂度解决滑动窗口问题。
```cpp
deque<int> q;

signed main() {
    cin >> n >> k;
    for (int i = 1; i <= n; i++) cin >> a[i];
    for (int i = 1; i <= n ;i++) {
        while (!q.empty() && q.front() < i - k + 1) q.pop_front();
        while (!q.empty() && a[q.back()] < a[i]) q.pop_back();
        q.push_back(i);
    }
}
```
